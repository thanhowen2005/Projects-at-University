INSTRUCTION OF HOW TO RUN SOURCE CODE: LET'S CHASE PAC-MAN

RUN BY USING VISUAL STUDIO CODE:
Step 1: You must install "visual studio code". You can follow this link: https://code.visualstudio.com/download

Step 2: Install python into your computer (You should use python version 3.12 or 3.13). You can follow this link: https://www.python.org/downloads/

Step 3: In "visual studio code", use key combination: Ctrl + Shift + X , then find python extension and install it.
Step 4: In your computer, open "Command prompt" then type this command line: "pip install pygame". This command line will help your python add packet pygame in order to run our code well.
Step 5: After unzip our project, our project's folder was named: 23127118_23127269_23127447_23127464

	In Visual Studio Code, using key combination: Ctr + K Ctrl + O (use must type at the same time) then select our folder and click into to folder named "Source", this will let you into our concept of project

	Besides using key combination, you can just use mouse clicking.
	On the toolbar in the top left corner, click to "File" then find "Open folder", after that just click to the folder that the same using key combination

Step 6: After all, click to file named "main.py" then using key combination: "Ctrl + Alt + N", this will help you run our project.
	In the other hand, you can run our project by use left mouse button and click to "Run Code". Our project will run.
