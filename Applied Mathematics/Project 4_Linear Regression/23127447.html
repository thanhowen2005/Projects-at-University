Phn a: Xy dng m hnh nh gi cht lng ru bng phng php hi quy tuyn tnh (s dng ton b 11 c trng)

Mc tiu

Mc tiu ca bi ton l xy dng mt m hnh hi quy tuyn tnh s dng phng php bnh phng ti thiu thng thng (Ordinary Least Squares - OLS) nhm d on cht lng ru da trn 11 c trng ha hc ca n.

Quy trnh thc hin

1. X l d liu

  • D liu bao gm nhiu mu ru, mi mu c m t thng qua 11 c trng ha hc v mt nhn quality (cht lng).

  • Bin quality l bin mc tiu (bin ph thuc), cn li l cc c trng u vo (bin c lp).

  • Tch bin mc tiu quality v cc c trng u vo.

  • Mi dng d liu c trng c thm mt phn t c gi tr 1.0 u, i din cho h s chn (intercept) trong phng trnh hi quy.

2. Xy dng cc cng c x l ma trn

  • trin khai hi quy tuyn tnh theo cng thc OLS, cn thc hin mt s php ton tuyn tnh:
    • Chuyn v ma trn: tnh $X^T$
    • Nhn ma trn: tnh $X^T X$ v $X^T y$
    • Nghch o ma trn: tnh $(X^T X)^{-1}$

3. Hun luyn m hnh hi quy tuyn tnh

  • S dng cng thc:

    $$ \beta = (X^T X)^{-1} X^T y $$

    tm ra vector trng s $\beta$, biu din nh hng ca tng c trng n cht lng ru.

4. Biu din m hnh

  • Sau khi tnh c cc h s $\beta$, biu din phng trnh hi quy tuyn tnh di dng:

    $$ Y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \dots + \beta_{11} x_{11} $$

  • Mi h s phn nh mc nh hng ca mt c trng n cht lng ru. C th nh gi c trng no quan trng hn da trn ln v du ca cc h s.

def mat_transpose(matrix):
    return [list(row) for row in zip(*matrix)]

def mat_mult(A, B):
    result = [[0 for _ in range(len(B[0]))] for _ in range(len(A))]
    for i in range(len(A)):
        for j in range(len(B[0])):
            s = 0
            for k in range(len(B)):
                s += A[i][k] * B[k][j]
            result[i][j] = s 
    return result

def mat_inverse(matrix):
    n = len(matrix)
    mat = [row[:] for row in matrix] 
    identity = [[float(i == j) for j in range(n)] for i in range(n)]

    for i in range(n):
        diag = mat[i][i]
        for j in range(n):
            mat[i][j] /= diag
            identity[i][j] /= diag
        for k in range(n):
            if i != k:
                factor = mat[k][i]
                for j in range(n):
                    mat[k][j] -= factor * mat[i][j]
                    identity[k][j] -= factor * identity[i][j]
    return identity

3. Hun luyn m hnh OLS

Hm train_ols(X, y) thc hin vic hun luyn hi quy tuyn tnh theo cng thc:

$ \beta = (X^T X)^{-1} X^T y $

Trong :

  • $ X $: ma trn c trng (c thm ct 1.0).
  • $ y $: vector gi tr cht lng ru.
  • $ \beta $: vector trng s (h s hi quy).
In[408]:
def train_ols(X, y):
    X_T = mat_transpose(X)
    XTX = mat_mult(X_T, X)
    XTy = mat_mult(X_T, [[v] for v in y])
    XTX_inv = mat_inverse(XTX)
    beta = mat_mult(XTX_inv, XTy)
    return beta

# Hm d on y_pred = X * beta
def predict(X, beta):
    return [sum(beta[j][0] * row[j] for j in range(len(beta))) for row in X]

# Hm tnh RSS
def rss_loss(y_true, X, beta):
    y_pred = predict(X, beta)
    rss = sum((yt - yp) ** 2 for yt, yp in zip(y_true, y_pred))
    return rss

import matplotlib.pyplot as plt

def plot_regression(x_vals, y_true, beta, feature_name, rss):
    """
    V biu  hi quy tuyn tnh cho 1 c trng
    - x_vals: list gi tr c trng (khng c bias)
    - y_true: list gi tr thc t ca y
    - beta: vector trng s [0, 1]
    - feature_name: tn c trng
    - rss: gi tr RSS tng ng
    """
    plt.figure(figsize=(6, 4))
    plt.scatter(x_vals, y_true, alpha=0.3, label='D liu thc')
    x_sorted = sorted(x_vals)
    y_line = [beta[0][0] + beta[1][0] * x for x in x_sorted]
    plt.plot(x_sorted, y_line, color='red', label='ng hi quy')
    plt.title(f"Hi quy tuyn tnh: {feature_name}\nRSS = {rss:.2f}")
    plt.xlabel(feature_name)
    plt.ylabel("Quality")
    plt.legend()
    plt.grid(True)
    plt.tight_layout()
    plt.show()

Phn c: Xy dng m hnh ring

Mc tiu

Mc tiu ca phn ny l xy dng mt m hnh hi quy tuyn tnh nhm tng kh nng biu din cc mi quan h phc tp gia cc yu t ha hc v cht lng ru.

Quy trnh thc hin

1. Chn c trng u vo (manually selected features)

  • La chn 11 c trng c nh.
  • Danh sch c trng c gi nguyn th t m bo tnh nht qun khi m rng v to c trng tng tc.

2. M rng c trng phi tuyn (Polynomial Expansion)

  • Vi mi c trng chn, to thm 5 c trng mi tng ng vi cc ly tha bc 2 n bc 6.
  • Vic thm ly tha gip m hnh hc c cc mi quan h phi tuyn gia c trng v bin mc tiu.

V d: nu chn c trng alcohol, ta to thm alcohol, alcohol, ..., alcohol.

3. To c trng tng tc (Interaction Feature)

  • Tnh tch ca ton b 11 c trng chn, to thnh mt c trng tng tc bc cao.

4. Thm h s chn (bias term)

  • Thm mt phn t c gi tr 1.0 vo u mi hng d liu c trng.
  • Cho php m hnh hc c h s t do (intercept) trong phng trnh hi quy.

Hun luyn v nh gi m hnh

1. Hun luyn bng phng php OLS

  • M hnh c hun luyn bng phng php bnh phng ti thiu (Ordinary Least Squares - OLS).
  • Tnh ton trng s hi quy $\beta$ da trn d liu c m rng.

2. Tnh sai s RSS

  • nh gi hiu qu ca m hnh bng cch tnh RSS (Residual Sum of Squares).
  • RSS th hin tng bnh phng sai s gia gi tr thc v gi tr d on.

Biu din m hnh hi quy

  • Ton b phng trnh hi quy c biu din di dng y vi tt c c trng:

    • 11 c trng gc
    • 55 c trng m rng (bc 2 n 6 cho mi c trng)
    • 1 c trng tng tc
    • Tng cng: 67 c trng + 1 bias

  • Phng trnh hi quy c dng:

    $$ y = \beta_0 + \sum_{i=1}^{11} \beta_i x_i + \sum_{i=1}^{11} \sum_{k=2}^{6} \beta_{ik} x_i^k + \beta_{int} \cdot (x_1 \cdot x_2 \cdot ... \cdot x_{11}) $$

Tng kt

M hnh trong phn ny c xy dng da trn c trng th cng, kt hp vi k thut m rng a thc v tng tc.
